# 🛡 SpamShield AI

An AI-powered web application that detects spam emails, phishing links, and malicious screenshots using Machine Learning, NLP, and OCR.

---

## 🚀 Features

- 📧 Email Spam Detection using Naive Bayes (NLP)
- 🔗 Phishing Link Detection (Rule-based)
- 🖼 Screenshot Spam Detection using OCR (Tesseract)
- 🎨 Interactive Web UI built with Streamlit

---

## 🧠 Technologies Used

- Python
- Scikit-learn
- Streamlit
- NLP (CountVectorizer)
- OCR (Tesseract)
